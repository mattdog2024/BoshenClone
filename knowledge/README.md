## 文件列表

### 1. 常老师实战操作体系详解.md
**核心内容：四层时区分工 + 完整操作流程**

- 日线 → 60分钟 → 15分钟 → 5分钟，四层时区分工
- 每一层都要形态成立才能切换到下一层
- 数据重合验证（三个时区目标位重合 = 概率极大）
- 1-3线安全，5-6线注意，7-8线危险（刀口舔血）

### 2. 日线到3线后的操作策略.md
**核心内容：日线到3线 ≠ 反转，是调整开始**

关键规则：
- **3线是调整区，不是反转区**
- 日线到3线 → 多单落袋为安，**直接切60分钟找做空机会**，不需要等日线形态
- 7线、8线才是反转区，才需要等日线形态确认
- 空单目标：60分钟2-3线就要出场（轻仓短空）
- 止损：设在日线3线上方合理位置

## 核心操作规则速查

| 日线到达位置 | 性质 | 正确做法 |
|---|---|---|
| 1-3线 | 上涨初期 | 逢低做多，安全区 |
| 3线附近 | 调整开始 | 多单离场，切60分钟找短空 |
| 5-6线 | 趋势中段 | 多单注意风险，轻仓 |
| 7-8线 | 反转高发区 | **必须等日线形态确认**才做空 |

## 60分钟分析规则

当日线到达3线后回调时：
1. 切换60分钟图
2. 从60分钟最近高点向下测量8条线
3. 判断当前在60分钟的几线
4. 等60分钟出现**向下形态**（右边K线低点 < 左边K线低点）
5. 形态成立后切换15分钟精确入场
6. 目标：60分钟2-3线（不贪心）
7. 止损：突破最近高点

## 形态确认规则

- **上涨形态**：右边K线高点 > 左边K线高点
- **下跌形态**：右边K线低点 < 左边K线低点
- 大周期形态没成立，小周期不能反向操作

## 测量规则（波神单体测量法）

- A点 = 基准K线的 Low（最低价）
- B点 = 同一根K线的 High（最高价）
- 振幅 = B - A
- 各线位 = A + 振幅 × 比例
- 比例（来自config.json）：[1.784, 2.351, 3.027, 3.459, 3.865, 4.622, 5.135, 5.865, 6.676]

## A/B点锁定规则

- 历史数据加载完成后，A/B点立刻锁定
- 锁定后每根新K线不重算A/B
- 只有价格走到8线位置，才解锁重新识别
