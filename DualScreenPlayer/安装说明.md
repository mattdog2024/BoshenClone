# 双屏播放器 - 安装与使用说明

## 系统要求

- **操作系统**: Windows 7 / Windows 8 / Windows 10（64位推荐）
- **Python**: 3.8 或更高版本（如果从源码运行）
- **VLC 播放器**: 必须安装（软件依赖 VLC 的解码引擎）
- **显示器**: 建议双屏（一个控制屏 + 一个大屏输出）

---

## 方法一：直接运行（推荐）

### 第一步：安装 VLC 播放器

前往 [https://www.videolan.org/vlc/](https://www.videolan.org/vlc/) 下载并安装 VLC 播放器（64位版本）。

**重要**：必须安装 VLC，否则软件无法播放视频！

### 第二步：安装 Python 依赖

打开命令提示符（CMD），运行：

```
pip install PyQt5 python-vlc Pillow mutagen
```

### 第三步：运行软件

双击 `run.bat` 或在命令提示符中运行：

```
python main.py
```

---

## 方法二：打包为 .exe 文件

### 第一步：安装 PyInstaller

```
pip install pyinstaller
```

### 第二步：运行打包脚本

双击 `build_windows.bat` 文件，等待打包完成。

打包完成后，可执行文件位于：`dist\DualScreenPlayer\DualScreenPlayer.exe`

## 软件使用说明

### 界面介绍

软件界面分为三个区域：

| 区域 | 位置 | 功能 |
|------|------|------|
| 播放列表 | 左侧 | 管理多个播放列表，可新建/重命名/删除 |
| 媒体素材区 | 中间 | 显示当前列表的所有媒体文件缩略图 |
| 预览控制台 | 右侧 | 预览下一个节目，显示节目列表 |

### 基本操作

**添加媒体文件**：点击"+ 添加媒体"按钮，或直接将文件拖拽到中间区域。

**播放媒体**：双击媒体卡片即可开始播放，视频会显示在大屏输出窗口上。

**控制播放**：使用底部控制栏的播放/暂停/停止/上一个/下一个按钮。

**黑屏**：点击工具栏"黑屏"按钮，或按 `Ctrl+B` 快速切换黑屏。

### 双屏设置

1. 将第二块屏幕（大屏）连接到电脑。
2. 点击工具栏"设置"按钮，在"输出设置"中选择输出屏幕。
3. 点击"输出屏幕"按钮，输出窗口会自动全屏显示在选定的屏幕上。

### 播放列表管理

- **新建列表**：点击左侧"+ 新建列表"按钮。
- **重命名**：右键点击列表名称，选择"重命名"。
- **切换列表**：点击左侧列表名称即可切换。

### 右键菜单功能

右键点击媒体卡片，可以：
- **播放**：立即播放此素材。
- **循环播放**：设置此素材循环播放。
- **设置为KV**：将此素材设为主背景视频（KV）。
- **属性设置**：设置音量、快捷键、播放完成动作等。
- **删除媒体**：从列表中移除此素材。

### 节目单保存/加载

- **保存**：`Ctrl+S` 或菜单"文件 → 保存节目单"，保存为 `.json` 文件。
- **加载**：`Ctrl+O` 或菜单"文件 → 打开节目单"。

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Space` | 播放/暂停 |
| `Ctrl+B` | 黑屏切换 |
| `Ctrl+S` | 保存节目单 |
| `Ctrl+O` | 打开节目单 |
| `Ctrl+N` | 新建节目单 |

## 支持的媒体格式

**视频**：MP4、AVI、MOV、MKV、WMV、FLV、RMVB、TS、M4V、3GP、WebM 等 99% 以上格式

**图片**：JPG、PNG、BMP、GIF、TIFF、WebP 等

**音频**：MP3、WAV、AAC、FLAC、OGG、WMA、M4A 等

## 常见问题

**Q: 视频播放没有声音？**
A: 检查工具栏音量是否为 0，或右键素材 → 属性设置 → 检查音量设置。

**Q: 输出窗口不在第二屏幕上？**
A: 点击"设置" → "输出设置"，选择正确的输出屏幕，然后点击"输出屏幕"按钮。

**Q: 软件启动报错？**
A: 确认已安装 VLC 播放器，且版本为 64位。

**Q: 视频卡顿？**
A: 对于 4K 视频，建议在素材属性中开启"解码优化"，或降低输出分辨率。
